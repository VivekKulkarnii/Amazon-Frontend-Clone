## 📌 Overview

A front-end recreation of Amazon's homepage built as a personal learning project. It covers the full page layout — navbar, hero banner, product category grid, and a multi-section footer — and adapts cleanly across desktop, tablet, and mobile screens.

---

## ✨ Features

| | |
|---|---|
| 🔍 | Navbar with search bar, sign-in, and cart |
| 🖼️ | Hero banner with overlay message |
| 🛍️ | 8 product category cards |
| 📱 | Fully responsive — desktop, tablet & mobile |
| 📋 | Multi-section footer with links and legal info |

---

## 🚀 Run Locally

Clone the repo and open `index.html` in your browser. No installation needed.

```bash
git clone https://github.com/VivekKulkarnii/Amazon-Frontend-Clone.git
```
