# Amazon Homepage UI Clone

A front-end replica of the Amazon homepage built using HTML and CSS, focusing on layout structure, visual styling, and UI similarity.

## Features
- Amazon-style navigation bar with logo, location, language, account, orders, and cart.
- Search bar with category dropdown and search icon.
- Hero banner with promotional message.
- Product category grid with background images.
- Multi-column footer with informational links.
- Hover effects and visual interaction highlights.

## Tech Stack
- HTML5
- CSS3 (Flexbox, gradients)
- Font Awesome Icons

## How to Run
1. Download or clone the repository.
2. Open `index.html` in any browser.
